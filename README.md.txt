# 🌍 AI Hackathon Project

This project was built for TCS AI Hackathon.  
We created an AI-powered app using **Streamlit + OpenAI/HuggingFace**.

## 🚀 Features
- **Mood Detector** 😀: Detects user mood and shows matching emoji.

## 🛠️ Tech Stack
- Python
- Streamlit
- OpenAI API / HuggingFace Transformers
